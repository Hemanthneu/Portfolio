Portfolio Projects

LLM - Medical Chatbot using LLaMA 2

Developed a sophisticated medical chatbot utilizing the LangChain library and the power of LLaMA 2 combined with Pinecone's vector store. This chatbot was built by extracting and processing data from the Gale Encyclopedia of Medicine using PyPDFLoader and DirectoryLoader. Text data was semantically embedded using Hugging Face's "all-MiniLM-L6-v2" model and stored in a Pinecone index for efficient semantic search capabilities. The chatbot's query responses are powered by a RetrievalQA chain from LangChain, with LLaMA 2 as the underlying language model.

MLOps - Credit card fraud detection

Developed a machine learning model to identify fraudulent transactions, leveraging a robust tech stack. Orchestrated with Apache Airflow, tasks such as data loading and preprocessing were automated. The model, built on RandomForestClassifier and monitored using MLFlow, stores data in GCP's cloud bucket via DVC. Deployment via Docker images, registered in Google Cloud's Artifact Registry, enables model hosting on Vertex AI. Regular bi-monthly retraining, managed by Airflow DAGs, ensures model adaptiveness to emerging fraud patterns, offering a reliable solution to mitigate credit card fraud.

NLP - Generating and scoring toxic comments

Toxicity Tune' employs a dual-model strategy, integrating HuggingFace BERT and miniGPT, to achieve an 85.44% accuracy rate in generating and scoring toxic comments. This was attained through fine-tuning optimization algorithms like Adam, SGD, AdaGrad, and LAMB, leading to a significant 55% reduction in model loss. The approach was further enhanced by innovative hyperparameter analysis and a robust 14-fold cross-validation methodology. This meticulous process allowed for the evaluation of 14 diverse models, maintaining loss values between 0.27 and 0.35, thereby demonstrating the robustness and effectiveness of the implemented strategies.

End-to-End ML Regression model

Pioneered the development of a end-to-end production-grade ML model leveraging a column transfer pipeline, achieving an extraordinary 95.3% accuracy, validated through meticulous R2 score comparison. Engineered with an arsenal of robust features including comprehensive logging, faultless exception handling, and swift data ingestion. Seamlessly integrating the Flask framework, it was encapsulated within a Docker image and deployed on both AWS Elastic Beanstalk and Azure Web App. This feat was elevated by establishing a seamless CI/CD pipeline orchestrated through GitHub actions, ensuring a continuous integration and deployment process that operates with impeccable precision. Furthermore, this system seamlessly interfaces with container registries, ensuring a flawlessly orchestrated deployment process

DeepFake Detection using Mesonet

Engineered a robust deep learning model utilizing the MesoNet convolutional neural network to achieve exceptional accuracy in detecting deepfake images. This entailed leveraging the MesoNet-4 architecture, integrating inverted residual blocks, and incorporating an attention mechanism to focus on critical features at a mesoscopic level. The model was meticulously trained on a bespoke dataset comprising 150,000 high-resolution images, generated through a combination of StyleGAN and Adobe Photoshop. To enhance efficiency, global unstructured pruning was applied at a sparsity of 20%, resulting in a lighter yet highly accurate model, boasting an impressive 97% accuracy in discerning deepfake content.
